## Backend

### Prerequisites

- Node.js installed 
- MongoDB installed and running
  
### Getting Started

1. Clone the repository
2. npm install
3. Create a .env file in the backend directory with the following content:

   ```bash
   
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret

4. npm start
